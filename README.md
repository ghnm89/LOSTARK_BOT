# LOSTARK_BOT
로스트아크 캐릭터 정보검색 BOT

## 명령어
* $help : 명령어 확인
* $정보 [캐릭터명] : 캐릭터 정보 검색
* $검증 [캐릭터명] : 캐릭터 레이드 커트라인 확인
* $시너지 [클래스명] : 클래스의 시너지 확인
  * 참고로 커트라인은 [원정대렙 200], [보석 7렙] 입니다. (재미로...)
